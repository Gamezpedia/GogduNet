Classpublic class SocketSecurity
InheritanceSocketSecurity Object

서버 클래스에서 허용 또는 비허용된 연결들의 목록을 가지는 클래스입니다.

여기서 허용되지 않거나 비허용된 연결의 클라이언트 측에선, 차단되어 연결이 끊겨도 GogduNetEvent.CONNECT_FAIL 이벤트가 발생하지 않습니다.

(즉, 연결을 경고나 알림 없이 바로 끊습니다)

(P2PClient에서만 특수하게, 객체에 추가할 Object 객체의 address 속성을 peerID로, port 속성을 음수로 설정해야 합니다.)

isPermission:Boolean

허용하는 목록인지 비허용하는 목록인지를 나타내는 값을 가져오거나 설정한다. true일 경우 이 객체의 목록들은 허용된 목록이며, false일 경우 이 객체의 목록들은 허용되지 않은 목록이 된다.
