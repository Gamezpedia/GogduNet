P2PClient()Constructor
public function P2PClient(url:String, netGroupName:String = GogduNet, socketSecurity:SocketSecurity = null, timerInterval:Number = 1000, connectionDelayLimit:Number = 10000)

url : 접속할 주소(rtmfp)

netGroupName : NetGroup 이름

socketSecurity : 통신이 허용 또는 비허용된 목록을 가지고 있는 SocketSecurity 타입 객체.

값이 null인 경우 자동으로 생성(new SocketSecurity(false))

timerInterval : 타이머 간격(ms)

(P2PClient의 timer는 정보 수신을 겸하지 않고 오로지 연결 검사용으로만 쓰이기 때문에 반복 속도(timerInterval)가 조금 느려도 괜찮습니다.)

connectionDelayLimit : 연결 지연 한계(ms)

(여기서 설정한 시간 동안 특정 피어로부터 데이터가 오지 않으면 그 피어와 연결이 끊긴 것으로 간주한다.)
